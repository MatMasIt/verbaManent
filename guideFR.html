VerbaManent

Introduction

VerbaManent est un logiciel de rédaction de procès-verbaux de réunion.

Il a été conçu pour rédiger les procès-verbaux des commissions scolaires et des comités d'école, mais il est adaptable à un grand nombre de situations.

Le logiciel vous permet d'organiser les agendas, les présences, les attributs de session et d'exporter les procès-verbaux au format PDF

VerbaManent est un logiciel libre: les utilisateurs ont la liberté d'exécuter, de copier, de distribuer, d'étudier, de modifier et d'améliorer le logiciel

Il est également distribué entièrement gratuitement, sous la licence open source GNU GPL 3.0 avec le code source entièrement disponible
