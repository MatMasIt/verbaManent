VerbaManent

Introduction

VerbaManent is a software for drawing up minutes of meetings.

It was designed to draw up minutes of school boards and school committees, but it is adaptable to a large number of situations.

The software allows you to organize agendas, attendance, session attributes, and to export the minutes in PDF

VerbaManent is libre software: users have the freedom to run, copy, distribute, study, modify and improve the software

It is also distributed completely free of charge, under the open source license GNU GPL 3.0 with source code fully available
